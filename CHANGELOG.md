# Changelog

All notable changes to Draft Coach will be documented in this file.

## 0.2.0 - Unreleased

- Add live and offline draft simulation, explicit target selection, category projection, and threat forecasting.
- Add optional professional-draft snapshots, provenance, freshness handling, analytics, and bounded scoring contributions.
- Add historical evaluation, calibration, resilience, performance, packaging, and Electron security coverage.
- Expand user, scoring, privacy, data-source, feedback, troubleshooting, and release-policy documentation.
- Refresh the Node 24 toolchain, GitHub Actions, dependency patches, and release metadata.

## 0.1.1 - 2026-07-04

- Ad-hoc sign macOS beta builds so Gatekeeper no longer treats the bundle as damaged because of an invalid resource seal.
- Document the remaining unsigned and unnotarized macOS beta install warning.

## 0.1.0 - 2026-07-01

- Prepare Draft Coach for a public beta release.
- Add release metadata, MIT license, public documentation, and CI/release workflow scaffolding.
- Document League client access, OP.GG/Data Dragon data usage, privacy expectations, and manual release gates.
- Keep the Phase 11 recommendation engine, explainability UI, composition readout, and settings presets as the first public feature set.
