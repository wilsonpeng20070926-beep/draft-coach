# Code of Conduct

Draft Coach uses the Contributor Covenant Code of Conduct, version 2.1.

Be respectful, constructive, and specific. Harassment, personal attacks, and bad-faith behavior are not welcome in project spaces.

Maintainers may remove content or restrict participation when conduct harms the project community.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
