# Third-Party Notices

Draft Coach depends on third-party open-source packages, including Electron, React, Vite, Vitest, league-connect, and the Model Context Protocol TypeScript SDK.

The packaged app should include notices generated from the dependency lockfile before each public release. At the time of release preparation, top-level runtime dependencies are permissively licensed, but maintainers must regenerate and review notices after dependency updates.

Data and game-asset attribution is documented separately in [docs/DATA_SOURCES.md](docs/DATA_SOURCES.md).
