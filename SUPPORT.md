# Support

Use GitHub Issues for public bug reports, feature requests, and troubleshooting once the repository is public.

Please include:

- Draft Coach version
- operating system
- League client region
- whether League was open before Draft Coach started
- what the status line said
- whether the issue happened in champ select, lobby, in game, or idle state
- screenshots with summoner names or private data removed

For security-sensitive reports, follow [SECURITY.md](SECURITY.md).
