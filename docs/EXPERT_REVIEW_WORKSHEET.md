## Structured judgments

- Was the active target correct? `Yes / No / Unclear`
- Did hover and lock transitions match the draft? `Yes / No / Unclear`
- Did any off-meta recommendation have enough visible justification? `Yes / No / N/A`
- Was any `Avoid` warning sufficiently specific and confident? `Yes / No / N/A`
- Did professional evidence add useful context rather than ornamental priority? `Yes / No / N/A`
- Did the recommendation preserve meaningful alternatives? `Yes / No`
- Was any wording more certain than the evidence supported? `Yes / No`
- Was any strategically important factor missing? `Yes / No`
- Did the result arrive in time to act? `Yes / No`

## Privacy and submission

- Participation is opt-in.
- Keep the worksheet local unless the reviewer explicitly chooses to submit it.
- Remove screenshots or notes containing account, chat, or teammate identifiers before sharing.
- Submit only through the project-maintainer channel listed in `docs/FEEDBACK_PROCESS.md`.
- A reviewer may request deletion of a submitted worksheet by its review date and assigned submission identifier.
